# Portifolio Ressurreicao

Landing page estática para apresentar as soluções com IA desenvolvidas por Pedro Ivo Florentino Ribas para o Escritório Ressurreição Advogados.

## Estrutura

- `index.html`: página principal
- `styles.css`: identidade visual e responsividade
- `script.js`: inicialização dos ícones
- `assets/`: imagem hero, perfis dos agentes, logo do CRM e QR Code do WhatsApp
